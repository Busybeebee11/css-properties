why CSS is important?

CSS known as Cascading Style Sheets as the name implies, are style sheets used and are considered to be highly effective tool to HTML that provides easy control over layout and presentation of website pages by separating content from design.
CSS is important for the following reasons;

  1. It controls how a web page looks and improves the website's presentation.
  2. Without it, websites would remain plain text on white backgrounds.
  3. It provides a set of guidelines by which a browser may place and render any element of HTML: a single page can be rendered in multitude of ways, whether stylistically or even functionally (for the color blind or the blind themselves).
  4. It is a building block of a website, as one can style a site by applying the various properties it has, which are used to present and change the skin of websites in different ways.
  5. It makes updates easier and smoother.

what is the difference between paddings, margins, and borders?

Paddings

Padding is the space between an element’s border and the element’s content. It shows the container background around the element in it. Padding can be affected by background colors as it clears the area around the content and does not allow negative values.

    Uses of paddings
  • Add space between content and its border.
  • Change the size of an element.
  • It controls the space inside an element and it determines how elements look and sit within a container.
Margins

A margin is the space around an element’s border. It controls the space outside an element and is completely transparent as it does not have any background color. It can take a negative or any float number

    Uses of margins
  • Change an Element’s position on the page.
  • Set the distance between nearby elements.
  • Overlap elements.
  • Move an element up or down on a page as well as left or right.
Borders

The border is any kind of line (solid, dotted, dashed...) surrounding the box, if present the border is the perimeter of the box. They can be invisible or thick colored lines. Borders can wrap immediately around an element, or it may look further away from an element if more padding has been applied to the element.

    Uses for borders
  • The CSS border properties allow you to specify the style, width, and color of an element's border.
  • It set visible border width to most HTML elements within the body.
